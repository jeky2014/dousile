= 逗死了
逗死了(dousile.com)，是一个经典的微信小笑话网站。每天发布最新最经典的爆笑小笑话、冷笑话、成人小笑话等，超级搞笑、爆笑不止。手机微信浏览，随时随地收获快乐，开心生活每一天。
=== 微信命令
* 来/来个/来一个/讲/讲个/讲一个：随机发送一个笑话
* 成人/黄色/黄的：随机发送一个成人笑话
* [其他……]：可以和你聊天
